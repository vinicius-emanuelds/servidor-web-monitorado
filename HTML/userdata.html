Configuração via User Data

Como configurar o servidor automaticamente ao iniciar a instância.

Passos:

1. Criar uma VPC com sub-redes públicas e privadas.

2. Criar uma instância EC2 na sub-rede pública.

3. Associar um Security Group permitindo HTTP e SSH.

4. Inserir o seguinte script no campo User Data:

#!/bin/bash
sudo apt update -y
sudo apt install -y nginx
sudo systemctl start nginx
sudo systemctl enable nginx
