Configuração via CloudFormation

Automatize a configuração da infraestrutura usando AWS CloudFormation.

Passos:

1. Criar um arquivo YAML com a definição da infraestrutura.

2. Subir o template no AWS CloudFormation.

3. Validar e iniciar o stack.

Exemplo de Template:

AWSTemplateFormatVersion: '2010-09-09'
Resources:
  MinhaInstancia:
    Type: AWS::EC2::Instance
    Properties:
      ImageId: ami-12345678
      InstanceType: t2.micro
      SecurityGroups:
        - !Ref MinhaSecurityGroup
